# ユースケース名
ログイン
# ユースケース概要
ログインする
# アクター
ユーザー
# 事前条件
- ログイン画面が表示されている
# 事後条件
- メニュー画面が表示されている
# 基本フロー
1. ユーザーは、ユーザーIDとパスワードを入力する
  1. ユーザーID、パスワードのいずれかが未入力の場合、代替フロー1
2. システムは、一致するユーザーの情報を取得する
  1. 一致するユーザーがいない場合、代替フロー2
3. システムは、「メニュー画面」を表示する

# 代替フロー
1. 入力画面に戻り、再入力を促すようメッセージを表示
2. 再度ログイン画面を表示し、「ユーザーが見つからない」と、メッセージを表示
